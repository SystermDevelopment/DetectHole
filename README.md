# 特発製作所 リング検出

## インストール手順

1. 仮想環境の立ち上げ
    ```cmd
    python -m venv venv
    ```

2. pipモジュールのインストール
    ```cmd
    python -m pip install -r ./requirements.txt
    ```

3. [main.py](./main.py) の実行

4. 【初回のみ】 PySimpleGUI のライセンスチェック とりあえず `Free Trial` をクリック
